== Description ==
A payment gateway that allows customers to pay using Crossmint.

== Installation ==
1. Upload `crossmint-plugin-gateway` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure Crossmint settings in WooCommerce -> Payments.
